## Installation

```bash
npm install
```
Important: Use npm 8.9.4 version to install some libraries

## Usage
Access http://localhost:3000/test to see the website

## SMTP Configuration
Update SMTP to your email address inside app.js


```bash
let smtpConfig = {
  service: 'gmail',
  auth: {
        user: 'xxxxxxxxx@gmail.com',
        pass: 'xxxxx'
    }
};
```
