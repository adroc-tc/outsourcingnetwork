## Installation

```bash
npm install
```

## Usage
Access `http://localhost:3000/test` to see the website

## SMTP Configuration
Update SMTP to your email address inside app.js


```bash
let smtpConfig = {
  service: 'gmail',
  auth: {
        user: 'xxxxxxxxx@gmail.com',
        pass: 'xxxxx'
    }
};
```
